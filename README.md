# NEON TRAILS

설치 없이 브라우저에서 실행되는 Three.js 기반 3D 오픈월드 자동차·바이크 게임입니다.

약 800 × 800 크기의 잔디 오픈월드에 레이싱 트랙, 점프장, 장애물 코스와 외곽 숲이 배치되어 있습니다.
레이싱 트랙과 외곽 초원에서는 6대의 AI 차량이 각자 경로를 따라 주행합니다.
맵 코인, 주행 거리, 접속 시간 보상으로 무료 게임 내 화폐인 크레딧을 획득할 수 있으며 잔액은 브라우저에 저장됩니다.

## 실행

`index.html`을 더블클릭하면 바로 실행됩니다. Three.js가 `vendor/three.min.js`에 포함되어 있어 인터넷 연결이나 별도 설치가 필요하지 않습니다.

선택 차량, 위치와 방향, 크레딧, 장신구, 보상 진행도 및 설정은 브라우저에 5초마다 자동 저장됩니다. 설정 메뉴의 `지금 저장` 버튼으로 즉시 저장할 수도 있습니다.

## 조작

- `W` 가속
- `S` 후진
- `D` 좌회전
- `A` 우회전
- `Shift` 강한 브레이크
- `Space` 드리프트
- `T` 전진 중 앞바퀴를 드는 윌리 묘기
- `V` 시작 지점 스폰 패드에서 선택 차량 소환
- `ESC` 일시정지 또는 메뉴 닫기
- 마우스 드래그 카메라 시점

## 메뉴

왼쪽 메뉴에는 요청한 다섯 항목만 있습니다.

1. 차고 — 자동차·바이크 선택 및 즉시 변경
2. 장신구 — 무료 컬러와 꾸미기 아이템
3. 순간이동 — 시작 캠프, 메가 점프장, 동쪽 레이싱 트랙, 북쪽 장애물 코스
4. 선물 — 접속 3·8·13·18·28·43·58분에 받는 무료 시간 보상
5. 설정 — 그래픽, 사운드, 감도, 전체 화면

유료 기능이나 결제 요소는 포함하지 않았습니다.

접속 시간과 남은 시간은 초를 버리고 줄이지 않도록 분 단위로 올림해 표시합니다.
